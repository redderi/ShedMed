Введение

SchedMed — это программа для медицинских учреждений, предназначенная для учёта медицинского оборудования и планирования поверочных событий. Она позволяет управлять базой данных оборудования, отслеживать калибровки и создавать отчёты.

Для кого: Работники, ответственные за инвентаризацию оборудования и соблюдение графика поверок.

Основные возможности:

  • Ведение базы данных оборудования (название, серийный номер, даты калибровки, местоположение).
  • Управление событиями калибровки (создание, редактирование, удаление).
  • Поиск оборудования по любым параметрам с поддержкой фильтрации.
  • Экспорт данных в форматы Excel и PDF, импорт данных из этих форматов.
  • Создание и восстановление резервных копий базы данных.

Важно: Все действия, изменяющие данные (удаление, импорт), требуют подтверждения для предотвращения ошибок.

Установка

Программа SchedMed поставляется в виде исполняемого файла или может быть запущена из исходного кода.

Запуск из исполняемого файла

  1. Скачайте установщик SchedMed с официального источника.
  2. Запустите установщик (SchedMedSetup.exe) и следуйте инструкциям.
  3. После установки запустите программу через ярлык на рабочем столе или в меню «Пуск».

Примечание: Убедитесь, что у вас есть права администратора для установки.

Интерфейс

Главное окно программы содержит следующие элементы:

Элемент Описание
Панель навигации Кнопки Главная, События, Резервные копии, Экспорт, Импорт, Руководство. Используйте их для переключения между списками оборудования, событий, функций экспорта/импорта, резервного копирования и открытия этого руководства.
Таблица оборудования На вкладке Главная. Отображает список оборудования. Заголовки таблицы автоматически переносятся на следующую строку при узких колонках благодаря кастомному отображению. Ширину колонок можно изменять мышью, размеры сохраняются между сеансами.
Таблица событий На вкладке События. Показывает список поверочных событий с переносом текста в заголовках.
Панель поиска Содержит поле для ввода запроса и выпадающий список для выбора колонки (например, «Наименование» или «Все колонки»). Поиск выполняется с небольшой задержкой для оптимизации.
Кнопки управления
  • Создать: Добавить новое оборудование.
  • Редактировать: Изменить данные выбранного оборудования.
  • Удалить: Удалить выбранное оборудование.
  • Создать событие, Редактировать событие, Удалить событие: Управление событиями на вкладке События.
  • Экспорт: Сохранить данные в Excel или PDF.
  • Импорт: Загрузить данные из Excel или PDF.
Контекстное меню Щёлкните правой кнопкой мыши на строке в таблице для доступа к опциям Изменить, Удалить, Отметить поверку сегодня (для оборудования).

Функции

Работа с оборудованием

  • Добавление оборудования:
    1. На вкладке Главная нажмите Создать.
    2. Заполните поля в открывшемся окне:
      • Наименование: Название оборудования (например, «Тонометр»). Обязательное поле.
      • Серийный номер: Уникальный номер. Обязательное поле.
      • Интервал калибровки:
        • Выберите из списка: «1 раз в 6 месяцев» (183 дня), «1 раз в год» (365 дней), «1 раз в 2 года» (730 дней), «1 раз в 5 лет» (1825 дней).
        • Или введите произвольное количество дней (например, «90»).
        • Поле обязательно, значение должно быть положительным числом.
      • Дата последней калибровки: Выберите дату в календаре или введите в формате ГГГГ-ММ-ДД (например, 2025-01-01). Обязательное поле.
      • Остальные поля (Тип, Местоположение, Номер свидетельства, Примечания) необязательны.
    3. Нажмите Сохранить. Новая запись появится в таблице.

    Рекомендация: Используйте предустановленные интервалы для стандартных поверок. Для нестандартных сроков вводите дни вручную, проверяя соответствие требованиям оборудования.

    Запрет: Не оставляйте обязательные поля пустыми. Не вводите отрицательные или нечисловые значения в поле интервала калибровки.

  • Редактирование оборудования:
    1. Выберите строку в таблице оборудования.
    2. Нажмите Редактировать или щёлкните правой кнопкой и выберите Изменить.
    3. Измените поля, затем нажмите Сохранить.

    Рекомендация: Проверьте серийный номер перед редактированием, чтобы убедиться, что выбрано правильное оборудование.

  • Удаление оборудования:
    1. Выберите строку в таблице.
    2. Нажмите Удалить или щёлкните правой кнопкой и выберите Удалить.
    3. Подтвердите удаление.

    Рекомендация: Убедитесь, что оборудование не используется, перед удалением.

  • Просмотр деталей:
    1. Дважды щёлкните по строке в таблице.
    2. Откроется окно с полной информацией об оборудовании.

    Рекомендация: Используйте двойной щелчок для быстрого просмотра.

  • Отметка поверки:
    1. Щёлкните правой кнопкой по строке и выберите Отметить поверку сегодня.
    2. Дата последней калибровки обновится на текущую дату, а следующая дата пересчитается.

Работа с калибровочными событиями

  • Добавление события:
    1. Перейдите на вкладку События.
    2. Нажмите Создать событие.
    3. Заполните поля.
    4. Нажмите Сохранить.
  • Редактирование события:
    1. Выберите событие в таблице на вкладке События.
    2. Нажмите Редактировать событие или выберите Изменить в контекстном меню.
    3. Измените данные и нажмите Сохранить.
  • Удаление события:
    1. Выберите событие.
    2. Нажмите Удалить событие или выберите Удалить в контекстном меню.
    3. Подтвердите удаление.

Поиск оборудования

  • Как искать:
    1. Введите запрос в поле поиска (например, «Тонометр» или «12345»).
    2. Выберите колонку в выпадающем списке (например, «Наименование») или оставьте «Все колонки».
    3. Результаты обновятся автоматически с небольшой задержкой.

    Рекомендация: Для точного поиска используйте серийный номер или ID.

    Запрет: Избегайте специальных символов (%, #) в запросе, чтобы не замедлить поиск.

Импорт и экспорт данных

  • Экспорт данных:
    1. Нажмите Экспорт на вкладке Главная.
    2. Выберите формат: Excel или PDF.
    3. Укажите папку и имя файла (по умолчанию: equipment_export_ГГГГ-ММ-ДД.xlsx/pdf).
    4. Нажмите Сохранить. PDF автоматически откроется после создания.

    Рекомендация: Используйте PDF для отчётов, Excel — для дальнейшей обработки данных.

    Запрет: Не сохраняйте файлы в системные папки (например, C:\Windows).

  • Импорт данных:
    1. Нажмите Импорт.
    2. Выберите формат: Excel или PDF.
    3. Выберите файл.
    4. Укажите, заменить текущие данные или добавить новые.
    5. Нажмите Сохранить.

    Рекомендация: Перед импортом создайте резервную копию базы данных.

    Запрет: Не импортируйте файлы с некорректными данными (например, неверные даты).

Резервное копирование

Резервное копирование защищает данные от потери. Создавайте копии перед импортом или удалением данных.

Создание резервной копии

  1. Нажмите Резервные копии в панели навигации.
  2. Выберите папку для сохранения архива.
  3. Нажмите Сохранить. Архив получит имя с текущей датой.

Рекомендация: Храните копии на внешнем носителе или в облаке.

Восстановление данных

  1. На вкладке Резервные копии выберите архив.
  2. Подтвердите восстановление.
  3. Перезапустите программу для обновления данных.

Рекомендация: Убедитесь, что текущие данные не нужны, так как они будут заменены.

Запрет: Не восстанавливайте архивы из неизвестных источников.
